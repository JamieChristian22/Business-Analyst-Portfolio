# Stakeholder Map

| Stakeholder Group | Examples (roles) | Needs / Concerns | Influence |
|---|---|---|---|
| Security / IAM | IAM Lead, Security Architect | Least privilege, auditability, policy enforcement | High |
| Compliance / Audit | Privacy Officer, Internal Audit | Evidence, control mapping, review cadence | High |
| RCM Operations | Billing Manager, Denials Lead | Faster access, fewer tickets, clear approvals | High |
| IT Apps / Integration | App Owners, Integration Team | Stable integrations, change windows, support | Med-High |
| HR / People Ops | HRIS Admin | Accurate worker data, term feeds, manager data | Medium |
| Service Desk | Tier 1/2 | Clear runbooks, fewer manual requests | Medium |
| End Users | Billers, Coders, AR reps | Frictionless requests, training | Medium |
| Leadership | VP Ops, CIO | Risk reduction, cost savings, measurable impact | High |
