# IAM Glossary (Healthcare RCM context)

- **IAM:** Identity and Access Management  
- **SailPoint:** Identity governance platform for lifecycle, access requests, and certifications  
- **JML:** Joiner / Mover / Leaver lifecycle events  
- **Birthright access:** Automatically assigned access for a role/department  
- **Requestable access:** Access requiring approval and time-bound assignment  
- **Access certification:** Periodic review where managers/app owners re-approve access  
- **SoD:** Segregation of Duties (prevent conflicting access combinations)  
- **PHI:** Protected Health Information (HIPAA)  
- **Least privilege:** Provide minimum access needed to perform job duties
